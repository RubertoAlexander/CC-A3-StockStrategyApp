Stock Analyst Web Application


About Us

Motivation

With a longstanding focus on financial markets, having tools which can make it easier to collate various technical and fundamental market analysis tools has been sought after. In order to find all relevant information for listed companies and their stock performance, various sites and resources are required. Further to this, there is no easy to use system that provides fully customizable technical trading strategy development and testing on past data.



Objective

To develop a hosted financial markets tool which provides the ability to:
- view individual stock information
- social media activity
- develop trading strategies
- back-test them and automate to generate alerts for the user
